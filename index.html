核心功能特性

安全隔离环境

基于 AWS Firecracker 技术，毫秒级拉起独立内核、完全隔离的微型虚拟机。每个任务都在独立虚拟环境中运行。

Git 集成

任务依赖代码仓库，拉取仓库项目分支、调用大模型生成代码、自动提交 PR 等系列流程。

并行智能体

同时启动多个智能体，分工合作，提高代码生成和任务处理效率。

云端运行

云端运行 Claude Code，无需本地配置环境，干净又高效。

第三方模型集成

友好界面

简洁直观的用户界面，支持文件上传、图片识别等功能。
